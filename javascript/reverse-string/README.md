# Reverse Number
Write a function called `reverseString` that accepts a string and returns a reversed copy of the string. Use `reverse-string.js` to provide your solution.

### Example 1:

Input: 'hello world'
Output: 'dlrow olleh'

### Example 2:

Input: 'asdf'
Output: 'fdsa'

### Example 3:

Input: 'CS rocks!'
Output: '!skcor SC'
